# DevOps

Categoria de prompts para operacoes da plataforma Aegis: Kubernetes, SRE, observabilidade, seguranca operacional e migracoes.

## Prompts

- [triagem-de-pods](triagem-de-pods/README.md): triagem de saude de pods Kubernetes com snapshot colado no prompt.
- [nota-de-triagem](nota-de-triagem/README.md): padronizacao de notas de alerta do Sentinel.
- [causa-raiz](causa-raiz/README.md): analise de causa-raiz a partir de artefatos operacionais.
- [backpressure-relay](backpressure-relay/README.md): apoio a decisao de estrategia de backpressure.
- [migracao-forge-event-driven](migracao-forge-event-driven/README.md): cadeia de prompts para migracao incremental do Forge.
- [networkpolicy-sentinel](networkpolicy-sentinel/README.md): geracao e refino de NetworkPolicy endurecida.
